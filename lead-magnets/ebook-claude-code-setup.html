Capítulo 1
¿Qué es Claude Code?

Claude Code es la interfaz CLI (línea de comandos) de Claude AI que corre EN TU COMPUTADOR, no en el navegador.

Diferencias clave vs ChatGPT

ChatGPT: Abres el navegador, escribes, copias la respuesta, la pegas en Excel. Fricción manual.

Claude Code: "Automatiza esto" → Se ejecuta directamente en tu PC. Crea archivos, edita documentos, procesa datos. Cero copypaste.

La diferencia = no esperas, ejecutas.

¿Qué puedo hacer?

  • Procesar 500 filas Excel y generar un informe PDF
  • Crear carpetas, renombrar archivos, organizar documentos
  • Buscar información en tus documentos y crear resúmenes
  • Automatizar emails, mensajes, tareas repetitivas
  • Analizar datos en tiempo real y generar gráficos
✨ Lo importante: No necesitas saber programación. Claude lo hace por ti.

Capítulo 2
5 pasos de instalación

1 Descarga Claude Code

Ve a claude.com/claude-code. Descarga la versión para tu sistema operativo (Windows, Mac o Linux).

Nota: Si eres usuario macOS, también puedes instalar vía terminal:
brew install claude-code

2 Abre una carpeta de trabajo

Crea una carpeta en tu computador donde guardarás tus proyectos. Ejemplo: ~/Proyectos/IA-Automaciones

3 Abre Claude Code en esa carpeta

Dentro de la carpeta, ejecuta: claude-code

Se abrirá una interfaz en tu terminal. Ya estás dentro.

4 Conecta tu cuenta de Claude

Sigue el prompt de autenticación. Inicia sesión con tu email de Anthropic.

5 ¡Listo! Tu primera tarea

Escribe: "crea un archivo README.md con mi nombre"

Si ves un archivo README.md creado en tu carpeta → ¡Funcionó!

✅ Success: Acabas de automatizar tu primera tarea.

Capítulo 3
3 tareas para empezar

Tarea 1: Organizar una carpeta desordenada

Instrucción:
"Tengo 200 archivos en ~/Descargas. Crea subcarpetas por tipo (PDFs, Excel, Imágenes, Otros) y mueve cada archivo a su carpeta."

Tarea 2: Extraer datos de Excel

Instrucción:
"Tengo clientes.xlsx. Extrae nombres + emails + teléfono. Crea un CSV limpio. Elimina duplicados."

Tarea 3: Generar un informe PDF

Instrucción:
"Toma los datos de mis ventas (archivo: ventas.csv). Crea un informe PDF con gráficos de ingresos por mes y resumen ejecutivo."

Con estas 3 tareas, habrás ahorrado 5-8 horas la primera semana.

Próximos pasos

1. Descarga Claude Code ahora: claude.com/claude-code

2. Crea tu carpeta de trabajo

3. Completa la instalación (5 pasos, 10 minutos)

4. Ejecuta una de las 3 tareas arriba

5. Piensa en TUS procesos manuales y automatiza los top 3
